Screenpipe is an open-source, 24/7 media capture tool designed to extract and process screen and audio data seamlessly. It provides a reliable pipeline for developers and end-users to manage data from screen recordings, audio inputs, and transcriptions. Built using Rust, with cross-platform support (Windows, macOS, and Linux), it is versatile and secure, allowing users to own their data while integrating personalized AI tools. Screenpipe serves as a self-hosted alternative to commercial solutions like Rewind.ai, offering flexibility and complete user control over data.

</MotionDiv>

<MotionDiv delay={0.5}>

## Key Features

- **24/7 Media Capture**: Captures screen and audio data continuously, storing it locally.
- **Personalized AI**: Enables AI models to be powered by your captured data.
- **Open Source & Secure**: Your data stays private, with complete control over storage and processing.
- **Cross-Platform**: Works on Windows, macOS, and Linux.
- **Multi-Monitor Support**: Supports multiple monitors for comprehensive data capture.
- **Plugins & Pipes**: Allows the creation and use of plugins (pipes) to extend functionality.

</MotionDiv>

<MotionDiv delay={1}>

## Target Audience

Screenpipe is suitable for developers, media professionals, and anyone interested in automating data capture and creating AI-powered workflows. Users can use it for tasks such as meeting transcription, screen activity logging, or even live captions during presentations.
